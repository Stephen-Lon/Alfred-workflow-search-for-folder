# Alfred-workflow-search-for-folder
A simple Alfred workflow to search for a folder
# Usage

Type `fs` (or the other keyword you may have set in User Configuration) followed by space and all or part of the folder name for which you wish to search. Pressing `Enter` will reveal the contents of the folder in Alfred. Pressing `âŒ˜ + Enter` will reveal the contents of the folder in Finder.
